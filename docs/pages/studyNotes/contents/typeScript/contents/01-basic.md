## 什麼是 TypeScript

簡單來說就是把 javascript 變成強語言的內容，讓你在寫程式時，可以更安全，更穩定，接到的 api 內容也可以更明確的知道該需要什麼類型。

## 怎麼使用 TypeScript

第一步安裝，Node.js 需要 16 以上，這個自己上網找啊ＸＤＤ

第二步，安裝 TypeScript

```bash
npm install -g typescript
```

第三步，安裝 ts-node

```bash
npm install -g ts-node
```

要執行檔案的話就直接打，

ts-node 檔名.ts

這樣就可以執行了，
